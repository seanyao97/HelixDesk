# HelixDesk 🧬

> 精简版电子实验记录本 (ELN) — 基于 Python + PySide6 的桌面应用，专为生物医学实验室设计。

## 功能

- **📋 计划表** — 月历视图，展示每日实验和待办事项
- **🧪 实验管理** — 项目分组、实验卡片、步骤时间线、结果记录
- **🧫 材料库** — 标签式材料管理
- **⚙️ 设置** — 自定义数据存储路径，支持云同步备份
- **🔍 搜索** — 全文搜索实验、步骤、材料

## 快速开始

```bash
# 1. 安装依赖
pip install PySide6

# 2. 启动
python main.py
```

或直接双击 `start.bat`。

## 技术栈

| 层 | 技术 |
|---|------|
| GUI | PySide6 (Qt6) |
| 数据库 | SQLite3 |
| Python | 3.12+ |

## 数据存储

默认路径：`%APPDATA%/HelixDesk/helixdesk.db`

可在 **设置 → 选择文件夹** 中自定义路径（建议选 OneDrive/iCloud 等云同步文件夹实现自动备份）。
